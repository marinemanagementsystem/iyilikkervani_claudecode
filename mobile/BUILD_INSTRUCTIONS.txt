==========================================
  ANDROID APK BUILD TALİMATLARI
==========================================

ADIM 1: Android Studio'yu açın

ADIM 2: Android Studio'da Terminal'i açın
   - Menü: View > Tool Windows > Terminal
   - Kısayol: Alt+F12

ADIM 3: Terminal'de komutları çalıştırın:

   cd C:\dev\imobile\iyilikkervani_claudecode\mobile\android
   gradlew.bat assembleDebug

ADIM 4: Build tamamlandığında APK dosyası şu konumda olur:

   C:\dev\imobile\iyilikkervani_claudecode\mobile\android\app\build\outputs\apk\debug\app-debug.apk

==========================================
NOTLAR
- Android Studio terminali JAVA_HOME ve ANDROID_HOME değişkenlerini otomatik ayarlar.
- İlk build 5-10 dakika sürebilir; sonraki build'ler daha hızlıdır.
